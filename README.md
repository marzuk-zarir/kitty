# Kitty Landing Page

## First install dependencies
```sh
npm install
```

## To create a development build
```sh
npm run build:dev
```

## To create a production build
```sh
npm run build:prod
```

## To start the webpack dev server
```sh
npm start
```

## To add vendor prefix for old browsers
```sh
npm run add:prefix
```
